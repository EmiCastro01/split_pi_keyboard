## Split-Keyboard Firmware
This repository contains the firmware for a split keyboard. This firmware is based on a custom implementation. It is designed to work with the `RP-2040` microcontroller.

## Documentation
The documentation for the firmware can be found in the `docs` directory. It contains detailed information.

## Build 
Follow the instructions in the `docs/BUILD.md` file to build the firmware.

## Contributing
Contributions to the firmware are welcome. If you have any suggestions or improvements, please feel free to submit a pull request or open an issue.

### Branches

The main branch is the stable version of the firmware. The `development` branch is where new features or improvements are pushed to. 
If you want to contribute, please create a new branch from `development` named `feature/your-feature-name` and submit a pull request when your feature is ready.
Please create a ticket in the issue tracker to discuss your feature before starting work on it.
